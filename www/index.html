What is maxLik?

  • maxLik is an extension package for the "language and environment for statistical computing and graphics" called R.
  • maxLik provides tools for maximum likelihood (ML) estimations.

What can maxLik do?

(Likelihood) maximization using the following algorithms
  • Newton-Raphson (NR)
  • Berndt-Hall-Hall-Hausman (BHHH)
  • Broyden-Fletcher-Goldfarb-Shanno (BFGS)
  • simulated annealing (SANN)
  • Nelder-Mead (NM)

Where can I get maxLik?

  • If you have internet access, you can install the current version of maxLik from a CRAN mirror by typing at the R prompt: 
     install.packages( "maxLik" )
  • Download current version of maxLik:
    • Package source: maxLik_0.5-2.tar.gz
    • Reference manual: maxLik.pdf
  • The current version (including binaries for Mac OS X and MS Windows) is available also at CRAN (The Comprehensive R Archive Network): http://cran.r-project.org/web/packages/maxLik/index.html
